# NIMBY__W3C

This project is transpiler that parsing W3C stadard code into nimDSL code.

There is two components follonwing:

* CSS2NIMDSL : Parsing CSS files into nimDSL AST.
* HTML2NIMDSL : Parsing HTML files into nimDSL AST.

## Folder Structure

```
NIMBY__W3C
        `-- CSS2NIMDSL  
        `-- HTML2NIMDSL
```
